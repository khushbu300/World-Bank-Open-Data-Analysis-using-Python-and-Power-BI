## Overview

This project presents an interactive Power BI dashboard for analyzing global development indicators using World Bank open data across 7 domains â€” health, economy, poverty, trade, environment, technology and labour market.
The dashboard enables comparison of 200+ countries and regions across multiple years and supports trend-based analysis using 20+ development indicators.

## Objectives

- Compare development indicators across 200+ countries and 7 domains

- Analyze multi-year trends for key socio-economic metrics

- Identify top and bottom performing countries using ranking-based views

- Study the relationship between health expenditure and life expectancy

## Data Source

World Bank Open Data (public datasets)

Country metadata obtained through the World Bank API

## Domains Covered

Health, Economy, Poverty, Trade, Environment, Technology, Labour Market (7 domains)

## Key Features

- Interactive filters for country, region and year

- Comparative analysis across 7 domains and 20+ indicators

- Time-series visualizations covering multiple years of data

- Country ranking views to highlight top and bottom performers

- Relationship analysis between health expenditure and life expectancy

## Tools Used

Power BI Desktop, DAX, Python, Jupyter Notebook

## Note

All datasets used in this project are based on publicly available World Bank open data and are intended for educational and analytical purposes.
